A PNG library, for faster website development, easier asset management, and faster site load times

by Awesome™

This is a library of 1x1 .PNGs — an asset server for PNGs — available for calling from any website. Know how Google has their library of Javascript (so websites can use Google's version of jQuery and whatnot? It's like that, but for 1x1 PNGs.)

How is this useful?

The PNGs here are intended for background CSS rules, such as for border lines with columns, when you don't want to image to affect the box-model's width.

They have very far-future expiration dates, so your visitors won't have to download the images more than once. Also, since they're on our server, you benefit from browsers' concurrent download settings (meaning: they can download assets from our server while they download assets from your server at the same time).

How do I use it?

To use, simply add the following to your css:

div#column{background:url(http://charliepark.org/colors/ccc.png) 0 0 repeat-x}

What colors are available?

At the moment, just #ccc.

We'll add to the list below as we create them.
